# sat-utils
A bouquet of usefull functions to handle satellite images

Some libraries used : 
- Sliced Aided Hyper Inference [SAHI](https://github.com/obss/sahi) 
- Segment Anything Model [SAM3 Ultralytics](https://docs.ultralytics.com/models/sam-3/) 
- You Only Look Once [YOLO Ultralytics](https://docs.ultralytics.com/models/yolo26/)
